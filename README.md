# My House Apartment JP

Modern full-stack apartment booking platform for showcasing and managing apartment reservations, reviews, amenities, and apartment information.

---

# Tech Stack

## Frontend

* React
* TypeScript
* Vite
* Tailwind CSS
* Component-based UI architecture

## Backend

* Node.js
* Express.js
* REST API architecture

## Database

* PostgreSQL

---

# Project Structure

```txt id="qup6hj"
my-house-apartments-jp/
├── backend/      # Express API + PostgreSQL backend
├── frontend/     # React frontend application
```

---

# Frontend Responsibilities

The frontend handles:

* Apartment pages
* Reservation UI
* Booking calendar
* Apartment galleries
* Sticky reservation sidebar
* Responsive layouts
* Internationalization (i18n)
* Reusable UI components
* Navigation and scrolling behavior

---

# Backend Responsibilities

The backend provides:

* Reservation management
* Apartment data APIs
* Availability validation
* Reservation conflict checks
* Review APIs
* Amenities APIs
* Location APIs
* Database communication

---

# Database Responsibilities

PostgreSQL is used as the main data source for:

* Apartment information
* Reservations
* Reviews
* Amenities
* Location data
* Apartment availability
* Apartment images and metadata

Planned database entities include:

```txt id="xw6q7t"
apartments
reservations
reviews
amenities
apartment_amenities
locations
images
```
